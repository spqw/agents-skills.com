How it works

Three steps. No setup wizard. No config files.

1

Work like you normally do

Open the app and start working. Agents Skills runs in the background and watches the patterns in your workflow — the clicks, the copy-pastes, the file moves.

2

It builds a skill

When it spots a repeatable pattern, it creates a "skill" — a reusable automation. You review it, tweak if needed, and approve.

3

Let it handle the rest

Next time that task comes up, trigger the skill and watch it run. Or set it to run automatically. You just got a coworker.
